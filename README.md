# bennett-eghan

my personal website.

**website:** https://www.bennett-eghan.com  
**framework:** react (vite)  
**deployment:** vercel  
**content:** mdx posts  
**styling:** tailwind css  

## running locally

```bash
$ git clone https://github.com/abena07/bennett-eghan.git
$ cd bennett-eghan
$ yarn install
$ yarn dev
```
